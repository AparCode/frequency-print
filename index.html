FrequencyPrint

About

FrequencyPrint is a tool that analyzes given audio files and identifies whether the file is deepfake-generated. This can range from full-on generated audio to AI covers (those that have real instrumentals but are sung by an AI).

The main purpose of FrequencyPrint is to utilize AI to distinguish covers that have deepfake-generated vocals from actual music covers. This is done by using Convolutional Neural Networks (CNNs) to detect artifacts inaudible to the human ear and not seen in real audio, using spectrograms of the audio files, making it easier for the program to classify audio.
